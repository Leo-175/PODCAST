# PODCAST
Projeto PODCAST com IA

Projeto com o objetivo de gerar um ebook digital com as facilidades das ferramentas de IA. todos os prompts seguem abaixo.
    ChatGPT - nome do podcast e roteiro
    gemini - capa do podcast
    ElevenLabs
    Capcut
